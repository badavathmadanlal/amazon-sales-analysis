Amazon Sales Data Analysis

This project focuses on analyzing Amazon sales data to uncover key business insights such as regional performance, product trends, and profitability. 
The analysis is visualized using an interactive Power BI dashboard.

Objectives

* Analyze overall sales performance
* Identify top-performing regions and segments
* Understand profit distribution across categories
* Discover trends and patterns in sales data

🛠️ Tools & Technologies

* Power BI
* Microsoft Excel
* Data Visualization Techniques

📂 Dataset
The dataset contains sales-related information including:

* Region
* Category & Sub-category
* Sales
* Profit
* Quantity

Dashboard Features

* Sales by Region (Central, East, South, West)
* Sales by Segment (Consumer, Corporate, Home Office)
* Category-wise performance
* Profit analysis
* Interactive filters and slicers

Key Insights

* The **West region** shows the highest sales performance
* Technology category contributes significantly to revenue
* Certain sub-categories generate high sales but low profit
* Seasonal trends impact overall sales distribution
